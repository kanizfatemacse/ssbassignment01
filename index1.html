HTML TAG

SI. Tags Description
01. ! Defines a comment
02. !DOCTYPE Defines the document type
03. a Defines a hyperlink
04. abbr Defines an abbreviation or acronym
05. address Defines contact information for the author
06. area Defines an area inside an imagr-map
07. b Defines bold text
08. del Defines text that has been deleted from a document
09. p Paragraph Tag
10. h1 to h6 Defines HTML headings
11. br Inserts a single line break
12. hr Defines a thematic change in the content
13. i Defines a part of text in an alternate voice or mood
14. mark Defines marked/highlighted text
15. strong Defines important text
16. sub Defines subscripted text
17. sup Defines subscripted text
18. u Defines text that should be stylistically different from normal text
19. var Defines a variable
20. frame Defines a window (a frame) in a frameset
21. iframe Defines an inline frame
22. img Defines an image
23. canvas Used to draw graphics, on the fly, via scripting
24. figure Specifies self-contained content
25. source Defines multiple media resources for media elements
26. Defines text tracks for media elements Paragraph Tag
27. link Defines the relationship between a document and an external resource (most used to link to style sheets)
28. nav Defines navigation links
29. ul Defines an unordered list
30. ol Defines an ordered list
31. li Defines an unordered list
32. dir Defines a directory list
33. dl Defines a description list
34. dt Defines a term/name in a description list
35. dd Defines a description of a term
36. menu Defines a list/menu of commands
37. table Defines a table
38. caption Defines a table caption
39. th Defines a header cell in a table
40. tr Defines a row in a table
41. thead Defines a cell in a table
42. tbody Groups the body content in a table
43. s Defines text that is no longer correct
44. tfoot Groups the footer content in a table
45. col Specifies column properties for each column within a element
46. colgroup Specifies a group of one or more columns in a table for formatting
47. style Defines style information for a document
48. div Defines a section in a document
49. span Defines a section in a document
50. script Defines a client-side script
